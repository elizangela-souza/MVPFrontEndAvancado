## Projeto Cooperativa de Reciclagem

>Este projeto é o resultado do MVP da sprint Desenvolvimento Front-end Avançado e buscou atender uma necessidade básica de uma cooperativa de reciclagem, que seria o armazenamento e consulta das informações referentes ao resultado do processo de triagem. 

Para isso, foi desenvolvido:
- Aba inicial com orientações para o usuário;
- Formulário para a inclusão de registros de cooperado e de triagem;
- Formulário de edição de registros de cooperado e de triagem;
- Tabelas de consulta das tabelas Cooperado e de Registro Triagem.

## Funcionalidades

- Inclusão, atualização, consulta e exclusão de registros;

## Como executar 

### 1. Utilizar o comando no terminal `npm start`
Para executar o aplicativo em modo de desenvolvimento.
Abra http://localhost:3000 para visualizar no seu navegador.

A página será recarregada automaticamente quando você fizer alterações.
Você também poderá ver quaisquer erros de lint no console.

### 2. Utilize o comando no termina `npm run backend`
Para executar o json-server, utilizado nesse projeto para simular as requisições no back-end.
Abra http://localhost:5000 para visualizar no seu navegador.
